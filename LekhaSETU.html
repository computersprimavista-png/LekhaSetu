🛡️ Data Backup & Restore

📍 Where is your data saved?

✅
What is safe:
  • Closing the browser
  • Restarting the computer
  • Moving the file to another folder
⚠️
Data will be lost if:
  • Browser history/cache is cleared
  • Browser is uninstalled
  • Opened on another computer
  • Different browser is used
💡 Therefore: Take a backup at least once a week and save the file in a safe place!

⬇️ Data Backup (Export)

Save your entire data to a file. This file will be downloaded to your computer.

Backup will include:
✅ Item Master  |  ✅ Customers/Parties
✅ Quotations  |  ✅ Sales Invoices
✅ Purchase Invoices  |  ✅ Payment Receipts

File: buddha_backup_[date].json

⬆️ Data Restore (Import)

Upload a previously saved backup file. Warning: current data will be replaced!

⚠️ Before restoring:
• Take a fresh backup first
• Use only your own backup file
• Only .json files will work

Only .json backup files accepted

💡 Backup Tips

📅 Take a backup at least once a week
💾 Save the backup file in Google Drive / Pen Drive as well
📱 You can also send the backup file on WhatsApp (to yourself)
🔄 To use app on a new computer - take backup first, then restore there
📁 Do not rename the backup file - the date is in its name

🗑️ Danger Zone

This option will permanently delete all data. Take a backup first!
